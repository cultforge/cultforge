Engineered for Excellence.

CultForge completely overhauls the launcher experience. Moving away from bloated web-wrappers, our bifurcated architecture guarantees safety, speed, and reliability.

Rust-Powered Client

The desktop application is built purely on Rust. Enjoy systems-level performance, memory safety, and a highly polished, instantly responsive user interface.

Smart Environment Detection

Offline-first design. If CultForge detects a localized, ready-to-play Minecraft environment on your machine, it intelligently bypasses unnecessary Mojang API calls.

Two-Tier Authentication

We separate social identity from game assets. Use Google OAuth for your CultForge community presence, and link your Xbox account exclusively to pull Microsoft assets safely.

FastAPI Infrastructure

Our backend relies on Python FastAPI deployed across four dedicated endpoints (auth, api, cdn, invite) ensuring max uptime and rapid content delivery.

Modrinth Integrated

Seamless, native integration with Modrinth's API. Search, install, and update modpacks, standalone mods, resource packs, and shaders directly from the UI.

Hybrid Open-Source

We operate on trust. The majority of the application logic is fully open to the community, while our critical encryption logic remains securely closed-source.

Choose Your Experience.

We distribute CultForge in two specialized versions to fit exactly how you prefer to play.

CultForge User Edition

A streamlined, lightweight client focused purely on game management. Stripped of extra social integrations for maximum focus and speed.

RECOMMENDED

Community Edition

The complete experience. Combines the base launcher with integrated, Discord-style community features, messaging, and seamless friend sync.
